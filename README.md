The application is built off a handmade API backend using Flask of popular cameras, while the front end is maintained by react and redux. Digitize is a basic modern retail website for cameras.

# App Description
-   Full stack application utilizing state management and a handmade custom API backend.  
-   A goat clone look-alike but with a camera spin.  
-   A working cart with redux persists.  

# Define Database Functionality + Input Data
1.  Redux persist stores User info and cameras added to cart.  
2.  User authentication provided by csrf and JWT token.  
3.  Search querying done but using redux state managament and running a filter method on the array.
